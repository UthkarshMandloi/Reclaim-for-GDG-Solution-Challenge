# ♻️ Reclaim - GDG Solution Challenge

**Reclaim** is an innovative platform designed to tackle the global waste crisis through advanced AI and community-driven action. Powered by **Google Gemini AI** and **Firebase**, Reclaim transforms waste into valuable resources by giving users step-by-step upcycling ideas, creating a circular economy, and fostering a collaborative community.

**Watch Explanation Here:** [https://youtu.be/BspWFMgbHJ4](https://youtu.be/BspWFMgbHJ4)

## 🌍 About the Project

Reclaim reimagines waste management by harnessing the power of **AI**, **sustainable design principles**, and **community collaboration**. We don't just recycle. We reimagine, reinvent, and reclaim what others discard. Reclaim is a real-world product built to scale sustainable living and reduce environmental impact through gamification and shared knowledge.

---

## ✨ Key Features

### 1. 🤖 AI-Powered Waste Analysis
Leveraging the **Google Gemini AI API**, Reclaim analyzes images of discarded items and instantly generates creative, practical step-by-step upcycling guides.

### 2. 💡 Community Idea Hub
A gamified platform where users can share their own upcycling projects, earn points (50 pts for ideas, 5 pts for likes), and get inspired by others.

### 3. 📝 Sustainable Living Blog
A dedicated space for articles on zero waste, sustainable fashion, advocacy, and DIY projects, categorized for easy reading.

### 4. 🔥 Google Technology Stack
We leverage a suite of Google tools to power Reclaim's infrastructure:
- **Google Gemini API**: For intelligent waste-to-idea generation.
- **Firebase Firestore**: For real-time database management, user tracking, and gamification logic.
- **Firebase Hosting / GCP**: For reliable and scalable hosting.

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/UthkarshMandloi/Reclaim-for-GDG-Solution-Challenge.git
   cd Reclaim-for-GDG-Solution-Challenge
   ```

2. Open the project:
   Since the project relies on vanilla HTML, JS, and CDN for Tailwind CSS, you can simply open `reclaim.html` in your browser. Alternatively, use a local development server (like VS Code Live Server) for the best experience.

3. Firebase Setup:
   The project connects to a live Firebase instance. To run your own instance, replace the Firebase configuration object in the JavaScript files (e.g., `js/ideas.js`, `js/community.js`, `js/blog.js`).
